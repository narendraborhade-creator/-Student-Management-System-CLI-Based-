# Student Management System (CLI-Based)

A Python CLI-based Student Management System with add, view, update, and delete functionality using dictionary storage and menu-driven interface.

## Features
- Add new students
- View all students
- Update student details
- Delete students

## Requirements
- Python 3.x

## Usage

```bash
python student_management.py
```
